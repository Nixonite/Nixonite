The Straightforward Guide to Data Science

Allen

2015-03-30 12:37

Source

The best guide to getting into the field of data science has been a tiny repo on Github called "go" which can be seen here.

Although it is a thorough guide, there's too much noise and it's actually overwhelming to see so many different topics to study at once. I have now compiled a shorter and more straightforward guide to getting into data science without sacrificing theory or practice.

I present to you, the straightforward guide to data science.

    Math

  1. Get acquainted with 3 courses of calculus (differential, integral, multivariate) and make sure to not skim over partial derivatives and gradients.
  2. Learn about linear algebra. You don't need a proof based course, and a good book (any edition) for this is written by Gilbert Strang
  3. Understand differential equations with this book (any edition)
  4. Get some probability and statistics under your belt! I recommend "All of Statistics" by Larry Wasserman. This is a hard book, so it's okay to only read the first part of it on Probability (section I up to ~87 pages?) for now. You will however want to read the rest of it for list item #9.

    Code

  5. You should absolutely know how to code in Python or R! It doesn't matter which you start with because after you've mastered one, you should learn the other... eventually. Here's where you should learn Python, and here's where you should look to start learning R. Don't learn both at once since it will take away time from other studies.
  6. Learn how to use the basic scientific tools of the language. As a Python person, this would be numpy, scipy, matplotlib, and scikit-learn.
  7. Learn SQL from either a book or some tutorials.
  8. Before you can mess around with models, you have to clean the data. Read a book on data wrangling, like this one.

    Gentle Guide to Data Analysis

  9. Study these books:
    • Mastering Machine Learning with Scikit-Learn (uses Python)
    • Introduction to Statistical Learning (uses R)

    Hard Guide to Data Analysis

    10. Pick at least two:
    • Elements of Statistical Learning
    • Introduction to Machine Learning (Adaptive Computation and Machine Learning Series)
    • Machine Learning: The Art and Science of Algorithms that Make Sense of Data

    Pretty Pictures and More Tools

  11. Visualization: Learn how to use D3.js or an equivalent fancy-visuals tool.
  12. Learn the other language (i.e. if you started with Python, now learn R)
  13. Learn how to use MongoDB, and read a book on Hadoop.

Congrats you're highly employable now!
